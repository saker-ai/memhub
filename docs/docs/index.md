# Memoh Documentation

Memoh(/ˈmemoʊ/) is a multi-member, structured long-memory AI agent platform. Create bots with independent workspaces, persistent memory, tool access, and channel integrations across Telegram, Discord, Lark (Feishu), QQ, Matrix, Misskey, DingTalk, WeCom, WeChat, WeChat Official Account, Email, and Web.

Memoh ships in two forms:

- **Desktop** for personal and local use. The native app starts its own local server, embedded Qdrant, storage, and bundled CLI.
- **Server Deploy** for always-on, multi-user, multi-tenant, or channel-facing deployments.
